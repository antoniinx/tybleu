# tybl.eu

A mysterious and edgy website for the tybl.eu group. Features a dark theme with green accents, glitch effects, and a clean, modern design.

## Features

- Dark theme with green accents
- Glitch effect on the title
- Typing animation for subtitle
- Responsive design
- Noise overlay for atmosphere

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts (Orbitron)

## Setup

1. Clone the repository
2. Open `index.html` in your browser
